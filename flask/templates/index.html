Introducing a new type of digital object: the Canned Analysis.

The section below contains an overview of this new digital object, frequently asked questions, its key features, and its properties.

Frequently Asked Questions

What is a Canned Analysis?

A Canned Analysis is a pre-run analysis of a biomedical dataset by computational tool. It is stored on a website and can be easily accessed by any user.

Where are they indexed?

Information about Canned Analyses, relevant datasets, and tools is currently indexed in a database managed by the Ma'ayan Lab at the Icahn School of Medicine at Mount Sinai.

Where are they hosted?

Canned Analyses are hosted on websites maintained by external developers. We do not currently provide hosting services.

Key Features

Dataset

The first key feature of a Canned Analysis is a biomedical dataset. It is identified by a unique accession, and must be accessible through an online repository.

Tool

The second key feature is a computational tool. It must be able to perform an analysis on the specified dataset, and has its own website.

URL

The third key feature is the Canned Analysis URL. It is a hard-coded link to a webpage containing the results of the analysis run by the tool on the dataset.

Annotations

Canned Analyses can be annotated by adding a brief description and any number of relevant metadata tags or keywords.

Properties

Findable

Canned analyses relevant to the desired data can be easily found by using the embedded interfaces. They are available on dataset search result pages and dataset landing pages. Demos available here.
 

Accessible

Canned analyses can be rapidly accessed by clicking on the links provided in the embedded interfaces. They are displayed as small icons with the tool's logo.

Flexible

Canned analyses can support multiple data types, tool types and analysis types.

Shareable

Canned analyses can be easily shared by copy-pasting their URLs, or by embedding them as icons in websites. The interface provides access to both solutions.
Developed by the Ma'ayan Lab at the Icahn School of Medicine at Mount Sinai.
